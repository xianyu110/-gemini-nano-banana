# 🚀 Gemini MVP 快速启动

## 1️⃣ 一键启动
```bash
# 进入项目目录
cd gemini-nano-banana

# 安装依赖（如果还没安装）
npm install

# 启动服务
npm run dev
```

## 2️⃣ 访问地址
- **MVP 测试页面**: http://localhost:3000/mvp
- **原始页面**: http://localhost:3000

## 3️⃣ 测试步骤

### 使用 Web 界面：
1. 打开 http://localhost:3000/mvp
2. 输入描述，例如："描述这张图片的内容"
3. 选择一张图片上传
4. 点击"发送请求"
5. 等待响应（约10-30秒）

### 使用测试脚本：
```bash
node test-gemini-api.js
```

## 4️⃣ API 已验证
✅ **测试结果**: API 正常工作
- 模型：gemini-2.5-flash-image-preview
- 状态：可用
- 特点：需要图片输入

## 5️⃣ 示例请求
```javascript
// 文字 + 图片
{
  prompt: "在这张图片中添加一只可爱的羊驼",
  imageData: "base64编码的图片数据"
}
```

## 6️⃣ 关闭服务
在终端按 `Ctrl+C` 停止服务

---
💡 **提示**: 该模型必须包含图片输入，不支持纯文本请求。
